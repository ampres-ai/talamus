    Service description

    Talamus Memory is a skills-only plugin that provides instructions for using the local Talamus command-line tool. The plugin does not provide a hosted account, cloud memory service, or remote data store. Installing it does not install the Talamus runtime, start a server, read files, enable capture, or call a language model.

    The published plugin is designed for read-only local retrieval and inspection. Its instructions may help an agent search and recall notes, inspect provenance and history, explore relationships, diagnose a brain, and preview repository ingestion without applying changes.
